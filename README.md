This repository reproduces the baseline analysis of cross-ideological voting behavior among Supreme Court justices from the 2020 Term through the 2024 Term.

# Overview:

This project constructs pairwise justice voting data and estimates cross-ideological alignment rates using logistic regression with case-level clustered standard errors.

The analysis measures how often:

- Conservative justices vote on the same side as liberal justices
- Liberal justices vote on the same side as conservative justices
- Alignment includes both majority and dissent votes.

# Data:

The analysis relies on the **justice-centered dataset** from the Supreme Court Database (SCDB).

To replicate:

1. Download the justice-centered dataset from: http://scdb.wustl.edu/index.php
2. Place the dataset file in the root directory of this repository.
3. Ensure the file name matches: `SCDB_2025_01_justiceCentered_Citation.dta`
   
The raw dataset is not included in this repository.

Replication Instructions
To reproduce all results:

1. Open Stata.
2. Navigate to the repository root directory.
3. Run: `00_master.do`

This will:

- Construct pairwise datasets
- Estimate logistic regression models
- Generate dot-and-whisker plots
- Export figures to the `output/figures/ directory
- Run pairwise comparisons

# Methods Summary

- **Sample**: 2020–2024 Terms
- **Unit of analysis**: justice-pair vote
- **Alignment**: voting on the same side (majority or dissent)
- **Estimation**: logistic regression
- **Standard errors**: clustered by case
- **Visualization**: margins + marginsplot
